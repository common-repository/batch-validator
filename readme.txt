=== Batch Validator ===
Contributors: wpdprx
Tags: HTML, XHTML, markup validation, validator, w3c
Requires at least: 2.1
Tested up to: 2.2.2

This plugin performs a batch markup validation check over your entire WordPress website.

== Description ==

It functions as a frontend for the Markup Validator Web Service API on http://validator.w3.org. Stylesheets are checked as well.

Languages available:
+ english
+ deutsch(german)

== Installation ==

Three easy steps:
	
1. Download, unzip and upload to your WordPress plugins directory
2. activate the plugin within your WordPress administration backend
3. go to "Batch Validator" on dashboard level
